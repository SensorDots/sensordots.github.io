MappyDot Technical Specifications

Physical

  • Size: 13x17.8x2.6 mm
  • Operating Temp: -20 to +70oC
  • Weight: <1g

Power

  • Operating Voltage: 2.8V to 5V DC
  • Continuous Mode (Default Measurement Profile): 21mA @ 3.3V
  • Quiescent Current: 1.3mA @ 3.3V (LED off)

Connectors

  • 7x2.54mm Castellated 2.54mm Input Connectors (compatible with 2.54mm pin headers or SMD solder pads)
  • 7x2.54mm Castellated 2.54mm Output Connectors (compatible with 2.54mm pin headers or SMD solder pads)
  • 6x1.27mm ICSP SPI Programming Header

Sensor

  • Field of View: 25 degrees
  • 940nm VCSEL Class 1 Laser
  • Programmable Green LED

Performance

  • Accuracy: Typically 3% up to 1.2m in High Accuracy Mode - see accuracy section in datasheet for more details.
  • Distance: 30mm to 2meters
  • Measurement Rate: Up to 50Hz
